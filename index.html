Git 4th Repo

Git repository initialization

First time initialization has 6 steps

  1. git init
  2. git add .
  3. git commit =m"meaningful comments"
  4. git branch -M main
  5. git remote add origin https://github.com/mdabualam/git-forth-repo.git
  6. git push -u origin main

After initialization 3 steps required to update your codes

  1. git init
  2. git commit -m"meaningful comments"
  3. git push
